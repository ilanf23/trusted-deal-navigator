# Commercial Lending X

Multi-role commercial lending platform with portals for superadmins, employees, clients, and partners.

## Stack

React 18 + TypeScript + Vite, shadcn/ui + Tailwind, Supabase (auth/db/edge functions), Twilio Voice SDK, TanStack Query.

## Development

```bash
npm install
npm run dev       # Start dev server (port 8080)
npm run build     # Production build
npm run lint      # ESLint
```

## Deployment

Deployed on Vercel at `trusted-deal-navigator.vercel.app`.
